🛡️ ¿Qué es Guardian360?

Guardian360 es una aplicación privada e independiente de seguridad ciudadana. No está afiliada ni representa a ninguna entidad gubernamental, policial ni municipal.

📋 Información que recopilamos

  • Nombre y correo electrónico para crear tu cuenta
  • Ubicación GPS para enviar alertas de emergencia
  • Fotos tomadas con la cámara como evidencia de emergencias
  • Número de celular para comunicación de emergencias
  • Datos de audio para el asistente de voz (procesados localmente)

🎯 Cómo usamos tu información

  • Para enviar alertas de emergencia a tu red de seguridad
  • Para mostrar tu ubicación a familiares y agentes autorizados
  • Para gestionar asistencia laboral y control de equipo
  • Para enviarte notificaciones importantes de seguridad

📷 Cámara y Micrófono

La aplicación solicita acceso a la cámara para monitoreo familiar y para tomar fotos de evidencia durante emergencias. El micrófono se usa para el asistente de voz Guardian. No accedemos a ninguno sin tu permiso explícito.

📍 Ubicación

Usamos tu ubicación GPS para enviar alertas con tu posición exacta y para mostrar tu ubicación a personas que tú autorices. La ubicación en segundo plano se usa para el rastreo de agentes de seguridad en campo.

🔒 Compartir información

No vendemos ni compartimos tu información personal con terceros. Los datos solo se comparten con los usuarios que tú autorices dentro de la aplicación.

🔐 Seguridad

Todos los datos se almacenan de forma segura en Firebase (Google Cloud). Usamos autenticación segura para proteger tu cuenta.
